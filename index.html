Multiplayer

Bring your team.
Keep your keys.

Share a tunneled URL, mint a 4-word invite, scan a QR. Your friend or co-founder hops into the same agent army — without ever touching your accounts or filesystem.

  • Three Access Modes

    Brief reads & writes tasks. Code drives sessions in auto/plan — Bash off unless you allowlist it. Full is owner-equivalent, TTL-bounded. Enforced at route guards, CLI flag injection, AND PreToolUse hooks — three independent layers.

  • Invites & QR Codes

    One-shot invite codes in three flavors: 4-word EFF Long speakable, Crockford base32 compact, or base64url QR token. Single-use, atomic SQL burn, per-IP rate-limited. No accounts, no signup — just paste, redeem, code together.

  • Mobile + PWA + Push

    Add to home screen on iOS / Android. Web Push via VAPID delivers off-screen notifications. Catch-up briefings show what changed since you last looked. Drive your agents from the train.

  • Live Catch-Up Briefings

    Returning to a stale session? IVE summarizes events, git commits, and memory changes since you left into a 2–5 sentence prose digest. One model call. Mode-aware filtering for joiners.

Security

Ship fast.
Stay safe.

Supply chain scanning, isolated worktrees, sandboxed accounts. Security at every layer — not bolted on.

  • Supply Chain Shield

    AVCP intercepts pip/npm install and GitHub Actions. 9-step deep scan per package. Blocks malware before it lands.

  • Isolated Worktrees

    Each agent gets its own git worktree. No merge conflicts from concurrent development.

  • Auth Autopilot

    Hit your Claude Max quota? IVE catches the quota_exceeded event, picks the next OAuth account in LRU order, refreshes its tokens via headless Playwright, swaps the session, and keeps you coding. Per-account sandboxed HOME directories — no cookie cross-contamination.

  • Event Bus & Audit Trail

    30+ event types. SQLite persistence, WebSocket broadcast, webhook delivery. Full audit trail.
